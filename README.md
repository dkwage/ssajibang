# 977

<h4>원했던 기능</h4>
<ul>
<li>간편한 일정 데이터 입력
<li>단순한 ui
<li>하늘이도 쉽게 이용할 수 있는 ux
<li>firebase를 이용한 데이터 동기화 및 입출력
</ul>
